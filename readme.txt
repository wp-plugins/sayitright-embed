== Description ==

What this plugin does is, using shortcodes, build a pure HTML version of a SayitRight station that you can navigate through.

**Features:**

* Embed a SayitRight station
* Launch a SayitRight station within a lightbox

**Included languages:**

* English

== Frequently Asked Questions ==

None yet.

== Installation ==

1. Download the plugin and unzip it.
1. Upload the folder sayitright-embed/ to your /wp-content/plugins/ folder.
1. Activate the plugin from your WordPress admin panel.
1. Installation finished.
